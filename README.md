# kualitas_tidur
Notebook ini berisi analisis kualitas tidur menggunakan pendekatan data analysis dan fuzzy logic berdasarkan dataset tahun 2018–2025.

## Struktur
- `HASIL_1.ipynb` : Notebook analisis
- `data/sleep_quality_dataset.csv` : Dataset (±5 MB)
